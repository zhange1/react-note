###### React Hooks使用限制

**有哪些**

- 不要在循环、条件或嵌套中使用Hook
- 在React的函数组件中调用Hook

**为什么**

- 设计初衷：改进React组件开发模式
- 问题领域：
  - 组件之间难以复用状态逻辑
    - 高阶组件
    - render props
    - 状态管理库
  - 复杂的组件变得难以理解
    - 生命周期与业务逻辑耦合
    - 组件 关联部分难以拆分
  - 人和机器都容易混淆类
    - this：值捕获问题、类属性草案
    - 编译优化
- 方案的原理：not magic，just arrays
